# ⚔️ RPGMenuSimulator

## 📝 Descripción del Proyecto
Este es un simulador de juego de rol (RPG) desarrollado íntegramente en Java por consola. El proyecto fue diseñado como un desafío técnico para aplicar los fundamentos de la programación estructurada.

El simulador permite crear un personaje, entrenar para mejorar estadísticas, participar en batallas por turnos y gestionar un inventario, todo bajo un sistema de menús dinámicos.

## 🚀 Tecnologías Usadas
* **Lenguaje:** Java 17 (o superior)
* **Entrada de datos:** `java.util.Scanner`
* **Control de Versiones:** Git y GitHub

## 🛠️ Conceptos Técnicos Aplicados
Para demostrar el dominio de la lógica básica, el programa implementa:
- **Estructuras de repetición:** `do-while` (menú principal), `while` (submenú de entrenamiento), `for` (sistema de combate) y `for-each` (recorrido de inventario).
- **Control de flujo:** `switch-case` y condicionales `if-else` anidados.
- **Estructuras de datos:** Arreglos (Arrays) para el sistema de ítems.
- **Validación de tipos:** Control de errores en la entrada de datos del usuario.

## 🕹️ Cómo Ejecutar
Sigue estos pasos para probar el juego en tu máquina local:

1. **Clonar el repositorio:**

   ```bash
   git clone https://github.com/PabloDesk/RPG-Menu-Simulator-Javalimos-Java
   ```

2. **Navegar a la carpeta:**

   ```bash
   cd RPG-Menu-Simulator-Java
   ```

3. **Compilar el programa:**

   ```bash
   javac RPGMenuSimulator.java
   ```

4. **Compilar el programa:**

   ```bash
   java RPGMenuSimulator
   ```
